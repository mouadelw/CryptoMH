# CryptoMH Pro - Professional Cryptography Suite

![CryptoMH Pro](https://img.shields.io/badge/CryptoMH_Pro-v1.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-green.svg)

A **professional, optimized cryptography application** featuring 8 encryption algorithms and 5 cryptanalysis tools with a modern GUI and crash-free performance.

## Key Features

### Encryption Algorithms
- **César** - Classic shift cipher
- **César MOT** - Caesar with keyword
- **Transposition Simple** - Columnar transposition
- **Transposition Complexe** - Advanced transposition
- **Playfair** - Digraph substitution cipher
- **DES** - Data Encryption Standard
- **AES** - Advanced Encryption Standard
- **RSA** - Public-key cryptography

### Cryptanalysis Tools
- **César Analysis** - Automatic shift detection
- **Transposition Analysis** - Key length analysis
- **Playfair Analysis** - Frequency analysis
- **Playfair CSP** - Constraint satisfaction solver
- **RSA Analysis** - Educational factorization

## Quick Start

### For Users
1. **Download** `CryptoMH_Pro.exe` 
2. **Double-click** to run - no installation required
3. **Select algorithm** from the dropdown
4. **Enter text** and key (if required)
5. **Click operation**: Encrypt, Decrypt, or Analyze
6. **Enjoy** professional cryptography tools!
